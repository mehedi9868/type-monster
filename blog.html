Topic - 01

Local Storage Vs Session Storage

sessionStorage and localStorage both are browser features. They look like almost same. localStorage doesn't expire but sessionStorage is cleared when the page session ends. More ever sessionStorage has less storage capacity than localStorage. In short sessionStorage is temporary but localStorage is permanent unless deleted.

Topic - 02

Global Scope Vs Block Scope

Global scope is such a scope or area where we can declare a variable and get value anywhere of the particular code. thats why it's called global scope. on the other hand block scope are not open. if a variable declared inside of block scope we can not access it out side of that particular scope or area. in javaScript block scopes are wrapped with curly braces and value by default can not cross the block area. however we can access values by using return.
